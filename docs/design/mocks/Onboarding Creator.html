FLOW · BRANCHES

Linear path: 1 Auth → 2 Email/pw → 3 Username → 4 Display name + avatar → 5 DOB → 6 Permissions intro → 7 Location (granularity → pre-prompt → OS prompt) → 8 Notifications (pre-prompt → OS prompt) → 9 Cam/mic (pre-prompt → 2× OS prompts) → 10 Legal → 11 Success.

Branches: Social auth skips step 2. Under-18 at step 5 hits the same terminal age-gate screen as Viewer (no retry). Each permission step has a denied path: location-denied = banner + reduced discovery; cam/mic-denied = blocking screen with iOS Settings deeplink; notifications-denied = soft banner + continue. Exit-intent sheet (see Viewer file) interrupts back-out from steps 2 onward.

A/B note on cam/mic: design includes a "Ask me on first stream" skip path so the team can test deferred cam/mic prompting against in-onboarding. Conversion measured at first-broadcast attempt.

PERMISSIONS · PRE-PROMPT → OS PROMPT SEQUENCE

Pattern: Wrld pre-prompt sets context (icon + title + 2–3 reasons + primary CTA). Tapping the CTA fires the native OS dialog. If denied in the OS dialog, do not re-prompt — fall through to the post-denial state (banner downstream of onboarding, plus a settings deeplink). This protects the OS-level permission ledger; iOS in particular won't show your prompt twice.

Foreground only: all location pre-prompts request "While Using" — never "Always". A creator who needs background location for a multi-day stream can opt up later from Settings.

Combined cam + mic: single Wrld screen, two OS prompts in sequence. If the first is denied, the second still fires (OS-managed), and the post-denial banner enumerates which one is missing.

JURISDICTION DETECTION

Detected silently from device locale + IP before step 10. EU (GDPR): granular consent toggles for analytics + personalization, essential locked on. California (CCPA): "Do Not Sell or Share My Info" toggle, off by default. Default (US/ROW): three-link layout, single Agree & Continue. Canada (PIPEDA), UK (UK-GDPR), Brazil (LGPD): alpha uses the EU variant; localize copy in beta.
