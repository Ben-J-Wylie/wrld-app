# `docs/design/references/`

Visual reference material that informs WRLD's aesthetic without driving a
specific audit row. Mood-board imagery, type specimens, color references,
inspiration shots, photography, deco / brutalist examples, anything that
sets the vibe.

These files **are not** scanned by the sub-phase 12.2 inventory pass.
They inform token decisions (palette, type weights, contrast targets,
motion feel) and aesthetic judgment calls at the principle level. The
inventory pass only audits [../mocks/](../mocks/).

Naming is free-form — organise by kind or by feel, whichever helps Ben
find things later. Subfolders welcome (`moodboard/`, `type/`, `color/`,
`inspiration/`, etc.) — created on first real need, not pre-built.

If something here turns out to be a mock-in-disguise (a finished screen
frame masquerading as inspiration), move it to `../mocks/` and rename
per that folder's convention.
